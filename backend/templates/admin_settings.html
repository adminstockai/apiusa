WhatsApp URL Format Guide

The WhatsApp URL should follow this format:

https://api.whatsapp.com/send/?phone=PHONE_NUMBER&text=MESSAGE

Replace PHONE_NUMBER with the phone number (including country code, no + sign)
Replace MESSAGE with your pre-filled message (URL encoded)
