应用启动说明 / Application Startup Guide
========================================

✅ 自动初始化已启用！数据库和管理员账号会在启动时自动创建。
✅ Auto-initialization enabled! Database and admin account are created automatically on startup.

快速启动 / Quick Start：
----------------
1. 进入 backend 目录 / Enter backend directory：
   cd backend

2. 安装依赖（首次运行）/ Install dependencies (first time only)：
   pip install -r requirements.txt
   或 / or
   python3 -m pip install -r requirements.txt

3. 启动服务器 / Start server：
   python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

就这么简单！应用会自动完成以下操作：
That's it! The application will automatically:
✓ 创建数据库目录 / Create database directory
✓ 初始化数据库表 / Initialize database tables
✓ 创建默认管理员账号 / Create default admin account
✓ 配置默认设置 / Configure default settings

访问应用 / Access Application：
----------------
- 主页面 / Main page：http://localhost:8000
- 管理员登录 / Admin login：http://localhost:8000/admin/login
- API 文档 / API docs：http://localhost:8000/docs
- 健康检查 / Health check：http://localhost:8000/health

应用架构 / Application Architecture：
----------------
后端 FastAPI 服务器（端口 8000）同时提供：
Backend FastAPI server (port 8000) provides:
✓ 前端静态页面 / Frontend static pages (index.html)
✓ 静态资源 / Static assets (/static directory)
✓ API 接口 / API endpoints (/api/token/...)
✓ 管理员后台 / Admin panel (/admin/...)
✓ 其他页面 / Other pages (disclaimer, terms, privacy)

所有功能都通过一个服务器访问！
All features accessible through a single server!
